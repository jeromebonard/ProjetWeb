Détails techniques

Préambule

J'ai passé un temps considérable sur la réalisation de ce site internet.

  • 8 jours durant les vacances de Noël
  • J'ai travaillé régulièrement dessus durant le mois de janvier
  • Les 3 derniers jours avant la date butoir

J'ai utilisé les contenus de cours fournis et je suis allé voir la documentation supplémentaire. Je me sens désormais à l'aise avec les fonctionnalités de base en html et css. J'ai aidé certains collègues sans même consulter d'aide en ligne. Avant de commencer quoi que ce soit, j'ai gardé à l'esprit en permanence qu'il fallait valider régulièrement le code html et css sur le site du W3C. Je n'ai à priori aucune erreur que ce soit en HTML ou CSS. En revanche, je pense qu'à certains endroits mon code pourrait être optimisé. Aucun autre code que html et CSS n'a été utilisé. J'ai ensuite réalisé les étapes qui suivent.

Etape 1 : Faire le point sur le projet

Objectif :

Il s’agissait de définir la raison pour laquelle nous devions créer un site internet à savoir faire de la vente en ligne.

Les contraintes étaient fournies dans l'énoncé.

Référencement :

Étant donné que le site n'existait pas avant, et que les contraintes indiquaient une structure de site professionnel, je me suis attaché à l'idée que ce site était créé pour obtenir de nouveaux clients et pour cela il fallait faire connaître le site en lui donnant la visibilité nécessaire.
Je me suis donc concentré sur deux points :

  • Le référencement naturel (Position du site sur les moteurs de recherche) en m'appuyant sur les préceptes du SEO sans pour autant pouvoir vérifier in fine si cela fonctionnait.
  • Les réseaux sociaux : j'en ai choisi deux : Instagram et Facebook (sans créer de compte ou de page dédiée sur les dits réseaux)

Etape 2 : Le thème et le cahier des charges

J'ai d'abord choisi le thème du site que je voulais créer. J'ai ensuite réfléchi au thème couleur, le blanc m'a semblé pertinent, ne serait-ce que pour obtenir un rendu uniforme peu importe le navigateur. J'ai ensuite pris des feuilles de papier et des crayons pour créer l'ensemble des pages, le positionnement des éléments… J'ai démarré par la page d'accueil (index), pour y intégrer qui seront récurrents tout au long du site.

  • Logo du site
  • Barre de navigation
  • Barre de recherche
  • Logo de connexion
  • Logo du panier
  • Le corps de page
  • Le footer

Etape 3 : Réalisation du projet

J'ai commencé ensuite la création de la page d'accueil. J'ai gardé à l'esprit tout au long de la création du site que celui-ci devait être responsive. J'ai donc commencé par ajouter un méta dans le html et des media queries dans le css. Le menu est horizontal en mode écran et en icone burger sur mobile. Pour le menu burger j'ai utilisé une checkbox qui masque le menu quand elle est décochée et qui l'affiche quand on clique dessus. Je n'ai pas envoyé la checkbox à l'extérieur de la page pour la masquer, je lui ai attribué une hauteur et une largeur nulle. L'ouverture du menu se fait donc au clic sur l'icône burger.

Le code CSS est divisé en 3 parties, une partie pour les écrans classiques, une partie pour les écrans type mobile et une partie commune. Le code est commenté pour se repérer dedans.

Je suis parti du principe que les pages seraient découpées grâce au display grid. Puis le positionnement des éléments des pages se fera grâce au display flex en très grande majorité.

Vous trouverez des duplicatas des pages pour la navigation quand un élément est ajouté au panier. Il y a un petit badge sur le panier avec le nombre d'éléments qu'il contient sur chacune des pages quand un article a été ajouté.

Ainsi, l'impression d'une navigation réelle est simulée.
